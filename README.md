# Carbon
A operating system using homemade Unix/Linux like kernel
## Build
```
// build image
make
// clean build directory
make clean
// using gdb debug
make debug
// using bochs
make bochsdbg
// gdb
make bochsgdb
// if you want mount/umount image
make mount_image/umount_image
// if you want delete mount point
make delete_mountpoint
```
